# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-03-16

### Added

- Real-time upstream speech translation (mic → translated speech → virtual output)
- Real-time downstream subtitle translation (audio bus → scrolling subtitles)
- Support for 22 languages with bidirectional translation
- Multiple TTS voices including regional Chinese accents
- First-run setup wizard with device auto-detection
- Floating, draggable subtitle overlay with adjustable opacity and font size
- Bilingual and translation-only subtitle modes
- Downstream voice output with volume control
- Settings window with audio device, language, subtitle style, and shortcut configuration
- i18n support: Traditional Chinese (zh-TW) and English (en)
- A/B model comparison support (qwen3-livetranslate-flash-realtime / gummy-realtime-v1)
